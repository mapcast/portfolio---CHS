과외커넥터

팀 프로젝트

과외커넥터는 JSP 팀 프로젝트로 선생님과 학생을 1:1로 매칭시켜주는 특징적인 시스템을 가지고 있습니다.

다중 조건+다중 항목 검색과 개인 페이지에서 미리 강의를 보고 선생님을 선택 할 수 있는 기능이 지원되며
1:1 채팅과 랜덤 조회, PW로 비밀번호를 바꾸고 이메일로 전송하는 등의 다양한 기능을 넣은 웹 사이트입니다.

이 프로젝트에서 제가 담당한 부분은 검색 관련 db 설계 참여, 검색을 위한 sql 쿼리문 작성, pw찾기 메일, 각종 등록페이지, 마이페이지, 헤더, 메인화면 등이 있습니다.

현재 http://busanit4033.cafe24.com 에서 웹호스팅을 진행하고 있습니다.

- 사용 기술 -
JSP

LottoNavy

팀 프로젝트

LottoNavy는 R과 Jsoup의 두가지 웹 크롤링 방법을 사용한 Spring MVC 웹 프로젝트입니다.

최근 1년간의 당첨 정보를 실시간으로 동행복권 사이트에서 가져와서 분석하도록 되어 있으며

랜덤 복권 번호 추천과 최근의 데이터를 바탕으로 번호를 추천해주는 기능도 있습니다.

이 사이트에서 제가 작성한 부분은 랜덤번호추천/jsoup 크롤링을 통한 이 주의 번호 추첨입니다.

- 사용 기술 -
Spring
R
Bootstrap

로또에 관한 데이터를 R로 분석

랜덤한 6자리의 번호 추천

JSoup로 최근 반년간 가장 많이 당첨된 6개의 번호 읽어오기

한번에 20~50개의 웹페이지의 정보를 가져오는 작업은 로딩 시간이 꽤 걸립니다
